<objective>
Add profile compaction (zero-run stripping + zero-pair null-marking + per-spectrum WLS δmz model) to
the default lossy chunk path, and populate FAIMS ion-mobility columns. Profile `spectra_data` is
compacted; centroid `spectra_peaks` is untouched. `--lossless` and `--point` stay bitwise-L1.

Purpose: shrink profile output and structurally match the HUPO-PSI reference (which null-marks
aggressively), and surface Thermo FAIMS compensation voltage that the schema already reserves.

Output: a null-aware ENCODE pre-pass + δmz fit/reconstruct codec, the `mz_delta_model` and
`scan.ion_mobility_*` columns populated, a `--mzpeak-no-null-marking` opt-out, and unit + integration
tests proving near-lossless reconstruction, centroid integrity, bitwise-L1 preservation, and FAIMS.
</objective>

  <behavior>
    - SkipZeroRuns(float[] inten) returns kept indices: a zero at i is DROPPED iff
      `(prevWasZero OR acc empty) AND ((i<n-1 AND inten[i+1]==0) OR i==n-1)`; first zero after a peak
      and last zero before a peak are kept; every interior zero dropped; one trailing zero kept at EOF.
      On the verified small.mzML spectrum-0 pattern (19913 pts) it keeps exactly 13589.
    - IsZeroPairMask(float[] keptInten) returns bool[]: position i true iff
      `inten[i]==0 AND (prevWasZero OR (i<n-1 AND inten[i+1]==0))`. On the kept array of spectrum 0,
      2376 positions are true; verified pattern keeps the kept-idx-7/8 NULL pair.
    - SelectDeltaModel(double[] mz, float[] inten) fits on the FULL pre-strip arrays:
        deltas = diff(mz) aligned to mz[1:]; weights w = sqrt(ln(I+1)) sliced [1:];
        constant δ̂ = median-below-median(deltas), e_const = Σ(δ̂−δ)²;
        regression: rows where δ≤1.0, design X=[1,mz,mz²] over mz[1:], WLS via QR (rows×w, target×w),
          requires ≥3 selected rows else fall back to constant; e_reg = Σ(predict(mz)−δ)² over ALL deltas;
        emit [β0] iff e_const < e_reg/10, else [β0,β1,β2]; if no mz → null (None).
      Golden: uniform 0.0909-spaced synthetic → [0.09090909] (constant); √-spaced synthetic ⇒ length-3.
      Replicate spectrum-0 betas ≈ [-2.2113928e-08, 9.697546e-11, 6.0542286e-09] to ~6 sig-figs when
      fed the small.mzML spectrum-0 arrays (or a faithful synthetic that reproduces them).
    - Predict(double[] coeffs, double mz) = Σ coeffs[i]·mz^i.
    - FillNulls(double?[] mz, double[] coeffs): reconstruct null m/z. Singleton-flanked null:
      `realValue − predict(realValue)` for a leading flank, `realValue + predict(realValue)` for a
      trailing flank. (Optional: local-median-delta for multi-point runs — discretion per RESEARCH
      Open Question 1; the ≤1e-4 tolerance covers either path.)
    - Round-trip: strip+mask+DeltaEncode (existing codec) → DeltaDecode → FillNulls reconstructs:
      constant-model spectrum bit-exact; quadratic-model spectrum max |Δm/z| ≤ 1e-4; every kept
      non-null point bit-exact; apex/max-intensity point bit-exact.
  </behavior>
